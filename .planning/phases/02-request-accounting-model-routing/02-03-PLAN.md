  <action>
Modify `copilot-backend.ts` to integrate accounting:

1. Add imports from `./accounting/index.js`: `RequestTracker`, `BudgetGuard`, `BudgetExceededError`, `getStageMultiplierTier`, `getModelMultiplier`, `getMultiplierValue`, `loadAccountingConfig`, `AccountingConfig`.

2. Extend `CopilotSessionBackend`:
   - Add optional `accountingConfig?: AccountingConfig` field.
   - Add `setAccountingConfig(config: AccountingConfig)` method.
   - Store config for session instantiation.

3. Decorate session handle creation — in `createSession()` and `resumeSession()`, if `accountingConfig` is set:
   - Create `RequestTracker` instance for the session.
   - Create `BudgetGuard` instance with config and tracker.
   - Wrap the returned `BackendSessionHandle` with an accounting-aware proxy.

4. The accounting-aware wrapper's `send()` method:
   - Extract current `stage` from message context (or default to "unknown"). Determine from the message metadata or conversation context how the caller identifies the current stage.
   - Call `getStageMultiplierTier(stage)` to get the stage's tier.
   - Call `getMultiplierValue(tier)` to get the estimated cost.
   - Call `guard.check(estimatedCost)` **before** forwarding to the real `send()`. If it returns a `BudgetWarning`, emit it (log to stderr or store for later display).
   - Forward to real `send()` to get the response.
   - Call `tracker.record(model, stage, tier)` **after** successful completion. The model comes from response metadata or the configured model for this session.
   - Return the original response unchanged.

5. Expose `getTracker(): RequestTracker | undefined` on the backend for telemetry consumption.

Modify `sdk.ts`:
- In `createAgentSession()`, when `backend === "copilot"`:
  - Call `loadAccountingConfig()` to get accounting configuration.
  - Pass config to `CopilotSessionBackend` via `setAccountingConfig()`.
  - This is the initialization entry point per D-04.

Keep changes minimal — only add accounting integration points, don't refactor existing backend logic.
